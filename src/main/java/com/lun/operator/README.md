## 赋值 ##

=

取右值，把它复制给左值

**基本类型**存储了实际的数值，而并非指向一个对象的引用，所以在为其赋值的时候，是**直接**将一个地方的内容**复制**到了另一个地方。

对一个**对象**进行赋值操作时，我们真正操作的是对对象的**引用**。所以倘若“将一个对象赋值给另一个对象”，实际是将“**引用**”从一个地方复制到另一个地方。
